# How to Do Well in This Class

Here are some tips for succeeding in this class: 

- Do the assigned reading.
- Ask questions about the readings!
- Participate in class discussions and activities on time. 
- Do the homework on time.
- Carefully read the assignment prompts.  
- Ask questions about the assignment prompts! 
- Re-read the assignment prompts. 
- Ask me questions about your writing projects.
