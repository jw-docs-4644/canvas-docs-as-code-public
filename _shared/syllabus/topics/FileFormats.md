# File Formats

For work that is turned in online, you should submit it in the format asked for in the assignment prompt. In most cases, this will be .docx, file. If I can’t open the file, I’ll ask you to resubmit, and the assignment will be marked as incomplete until I can view the submission. 

Note that Google Docs and LibreOffice allow you to save your work in the docx format. 
In classes where we are learning about other formats, such as Markdown or DITA, the assignment prompt will indicate which format your work should be submitted in.
